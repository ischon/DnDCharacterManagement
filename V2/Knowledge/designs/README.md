# Designs - DnD Character Management (V2)

Deze map bevat de geëxporteerde design assets van het Stitch project.

## Bronbestand
De originele ontwerpen, inclusief de mogelijkheid om nieuwe schermen te genereren of bestaande aan te passen, zijn te vinden in het Stitch project:
[DnD Character Management - Stitch Project](https://stitch.withgoogle.com/projects/5554278943842960542)

## Inhoud
- **Screenshots**: Visuele weergave van de dashboards en componenten (`.png`).
- **HTML Content**: De bijbehorende broncode voor de ontwerpen (`.html`).

Voor toekomstige updates kunnen de schermen in Stitch worden aangepast en opnieuw naar deze map worden gedownload.
